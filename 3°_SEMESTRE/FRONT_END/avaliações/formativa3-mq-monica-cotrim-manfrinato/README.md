# GalaxyTours - Viagens Espaciais 🚀🧑‍🚀

Esse é um projeto que foi desenvolvido para a avaliação formativa de Front-end, focado em tornar uma página de viagens espaciais responsiva!

## 🛠️ Tecnologias Utilizadas
* HTML5 (Semântica e Acessibilidade)
* CSS3 (Estilização e Feedback Visual)

## 📁 Estrutura do Projeto
O projeto foi organizado seguindo boas práticas de mercado:

/
├── index.html
└── src/
    ├── styles/  (Estilos da página)
    ├── scripts/ (Lógica de interação)
    └── images/  (Ativos visuais)

## 🚀 Funcionalidades Implementadas
* **Responsividade:** Página adaptável ao seu dispositivo, seja notebook, tablet ou smartphone
